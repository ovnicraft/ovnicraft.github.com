La aplicación

Vamos a desarrollar una app de microblogging que se llamara microblog.
Los temas que se cubriran en el tutorial son:

  • Gestión de usuarios, incluye login, sesiones, roles de usuarios, perfiles.
  • Gestion de base de datos, incluye proceso de migracion.
  • Soporte de formularios web con validacion de campos.
  • Paginacion de items
  • Full text search
  • Notificaciones de coreos electronicos a usuarios.
  • Planitllas HTML.
  • Soporte para multiples lenguajes.
  • Caching y otros optimizaciones de rendimiento.
  • Tecnicas de depuracion para desarrollo y servidores de produccion.
  • Instalacion en un servidor de produccion.

    • Instalando Flask

    Para la siguiente instruccion debes instalar virtualenv, entonces:

    $ sudo pip install virtualenv

    Ahora tenemos un ambiente de desarrollo listo para trabajar.

    Vamos a crear un directorio llamado: microblog y dentro de este directorio ejecutamos:

    $ virtualenv flask
